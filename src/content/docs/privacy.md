## Overview

AI Security in Practice is a static documentation site. It does not collect, store, or process personal data directly.

There are no user accounts, no contact forms, no newsletter signups, no comments, and no tracking scripts.

## Cookies

This site does not set cookies. Client-side features (such as theme preference and sidebar state) use browser local storage or session storage, which is not transmitted to any server.

## Hosting

This site is hosted on [Vercel](https://vercel.com/). Vercel may process technical data such as IP addresses, browser type, and access timestamps as part of serving web pages. This processing is governed by [Vercel's Privacy Policy](https://vercel.com/legal/privacy-policy).

## Analytics

This site does not currently use any analytics service. If analytics are added in the future, this policy will be updated to reflect what data is collected and how it is processed.

## Search

The site search is powered by [Pagefind](https://pagefind.app/), which runs entirely in the browser. No search queries are sent to any external service.

## Your Rights

Under UK GDPR, you have the right to request information about any personal data processed in connection with your use of this site. Given that this site does not collect personal data, there is unlikely to be any data to disclose. If you have questions, you can contact the site maintainer via GitHub.
